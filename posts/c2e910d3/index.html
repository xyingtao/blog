SpringBootзӣёе…ій—®йўҳ

д»Җд№ҲжҳҜSpringBoot?дёәд»Җд№ҲдҪҝз”ЁSpringBoot?

  • з”ЁжқҘз®ҖеҢ–Springеә”з”Ёзҡ„еҲқе§Ӣжҗӯе»әд»ҘеҸҠејҖеҸ‘иҝҮзЁӢ дҪҝз”Ёзү№е®ҡзҡ„ж–№ејҸжқҘиҝӣиЎҢй…ҚзҪ®(propertiesжҲ–ymlж–Үд»¶)
  • еҲӣе»әзӢ¬з«Ӣзҡ„springеј•з”ЁзЁӢеәҸ mainж–№жі•иҝҗиЎҢ
  • еөҢе…Ҙзҡ„Tomcatж— йңҖйғЁзҪІwarж–Үд»¶
  • з®ҖеҢ–mavenй…ҚзҪ®
  • иҮӘеҠЁй…ҚзҪ®springж·»еҠ еҜ№еә”еҠҹиғҪstarterиҮӘеҠЁеҢ–й…ҚзҪ®

springbootжқҘз®ҖеҢ–springеә”з”ЁејҖеҸ‘,зәҰе®ҡеӨ§дәҺй…ҚзҪ®,еҺ»з№ҒйҮҚз®Җ,just run е°ұиғҪеҲӣе»әдёҖдёӘзӢ¬з«Ӣзҡ„,дә§е“Ғзә§еҲ«зҡ„еә”з”Ё

SpringBoot зҡ„дјҳзӮ№йқһеёёеӨҡ

  1. зӢ¬з«ӢиҝҗиЎҢ

    SpringBootеҶ…еөҢдәҶеҗ„з§Қservletе®№еҷЁ,Tomcat,Jettyзӯү,зҺ°еңЁдёҚйңҖиҰҒжү“еҢ…жҲҗwarеҢ…йғЁзҪІеҲ°е®№еҷЁдёӯ,SpringBootеҸӘиҰҒжү“еҢ…жҲҗдёҖдёӘеҸҜжү§иЎҢзҡ„jarеҢ…е°ұиғҪзӢ¬з«ӢиҝҗиЎҢ,жүҖжңүзҡ„дҫқиө–еҢ…йғҪеңЁдёҖдёӘjarеҢ…еҶ…

  2. з®ҖеҢ–й…ҚзҪ®

    Spring-boot-starter-webеҗҜеҠЁеҷЁиҮӘеҠЁдҫқиө–е…¶д»–з»„д»¶,еҮҸе°‘дәҶmavenй…ҚзҪ®

  3. иҮӘеҠЁй…ҚзҪ® SpringBootиғҪж №жҚ®еҪ“еүҚзұ»и·Ҝеҫ„дёӢзҡ„зұ»,jarеҢ…жқҘиҮӘеҠЁй…ҚзҪ®bean,еҰӮж·»еҠ дёҖдёӘspring-boot-starter-webеҗҜеҠЁеҷЁе°ұиғҪжӢҘжңүwebеҠҹиғҪ,ж— йңҖе…¶д»–й…ҚзҪ®

  4. ж— д»Јз Ғз”ҹжҲҗе’Ңxmlй…ҚзҪ®

    SpringBootй…ҚзҪ®иҝҮзЁӢдёӯж— д»Јз Ғз”ҹжҲҗ,д№ҹж— йңҖxmlй…ҚзҪ®ж–Үд»¶е°ұиғҪе®ҢжҲҗжүҖжңүзҡ„й…ҚзҪ®е·ҘдҪң,иҝҷдёҖеҲҮйғҪжҳҜеҖҹеҠ©дёҺжқЎд»¶жіЁи§Је®ҢжҲҗзҡ„,иҝҷд№ҹжҳҜSpring4,xзҡ„ж ёеҝғеҠҹиғҪд№ӢдёҖ

  5. еә”з”Ёзӣ‘жҺ§

    SpringBootжҸҗдҫӣдёҖзі»еҲ—з«ҜзӮ№еҸҜд»Ҙзӣ‘жҺ§жңҚеҠЎеҸҠеә”з”Ё,еҒҡеҒҘеә·зӣ‘жөӢ

SpringBootж ёеҝғжіЁи§ЈжҳҜе“ӘдёӘ,е®ғдё»иҰҒз”ұе“ӘеҮ дёӘжіЁи§Јз»„жҲҗзҡ„?

еҗҜеҠЁзұ»дёҠйқўзҡ„жіЁи§ЈжҳҜ@SpringBootApplication,е®ғд№ҹжҳҜSpring Bootзҡ„ж ёеҝғжіЁи§Ј,дё»иҰҒз»„еҗҲеҢ…еҗ«дәҶдёҖдёӢдёүдёӘжіЁи§Ј:

@SpringBootConfiguration: з»„еҗҲдәҶ@ConfigurationжіЁи§Ј,е®һзҺ°й…ҚзҪ®ж–Үд»¶зҡ„еҠҹиғҪ

@EnableAutoConfiguration: жү“ејҖиҮӘеҠЁй…ҚзҪ®зҡ„еҠҹиғҪ,д№ҹеҸҜд»Ҙе…ій—ӯжҹҗдёӘиҮӘеҠЁй…ҚзҪ®зҡ„йҖүйЎ№,еҰӮе…ій—ӯж•°жҚ®жәҗиҮӘеҠЁй…ҚзҪ®еҠҹиғҪ: @SpringBootApplication(exclude = { DataSourceAutoConfiguration.class})

@ComponentScan: Springз»„д»¶жү«жҸҸ

иҝҗиЎҢSpringBootжңүе“ӘеҮ з§Қж–№ејҸ?

  1. жү“еҢ…з”Ёе‘Ҫд»ӨжҲ–иҖ…ж”ҫеҲ°е®№еҷЁдёӯиҝҗиЎҢ
  2. з”ЁmavenжҸ’д»¶иҝҗиЎҢ
  3. зӣҙжҺҘжү§иЎҢmainж–№жі•иҝҗиЎҢ

еҰӮдҪ•зҗҶи§ЈSpringBootдёӯзҡ„Starters?

StartersеҸҜд»ҘзҗҶи§ЈдёәеҗҜеҠЁеҷЁ,е®ғеҢ…еҗ«дәҶдёҖзі»еҲ—еҸҜд»ҘйӣҶжҲҗеҲ°еә”з”ЁйҮҢйқўзҡ„дҫқиө–еҢ…,дҪ еҸҜд»ҘдёҖз«ҷејҸйӣҶжҲҗSpringеҸҠе…¶е®ғжҠҖжңҜ,иҖҢдёҚйңҖиҰҒеҜјеҮәжүҫе®һдҫӢд»Јз Ғе’Ңдҫқиө–еҢ….еҰӮжһңжғідҪҝз”ЁSpring JPAи®ҝй—®ж•°жҚ®еә“,еҸӘиҰҒеҠ е…Ҙspring-boot-starter-data-jpaеҗҜеҠЁеҷЁдҫқиө–е°ұиғҪдҪҝз”ЁдәҶ.StartersеҢ…еҗ«дәҶи®ёеӨҡйЎ№зӣ®дёӯйңҖиҰҒз”ЁеҲ°зҡ„дҫқиө–,е®ғд»¬иғҪеҝ«йҖҹжҢҒз»ӯзҡ„иҝҗиЎҢ,йғҪжҳҜдёҖзі»еҲ—еҫ—еҲ°ж”ҜжҢҒзҡ„з®ЎзҗҶдј йҖ’жҖ§дҫқиө–.

Starterе‘ҪеҗҚ

Spring Bootе®ҳж–№зҡ„еҗҜеҠЁеҷЁйғҪжҳҜд»Ҙspring-boot-starter-е‘ҪеҗҚзҡ„пјҢд»ЈиЎЁдәҶдёҖдёӘзү№е®ҡзҡ„еә”з”Ёзұ»еһӢгҖӮз¬¬дёүж–№зҡ„еҗҜеҠЁеҷЁдёҚиғҪд»Ҙspring-bootејҖеӨҙе‘ҪеҗҚпјҢе®ғд»¬йғҪиў«Spring Bootе®ҳж–№дҝқз•ҷгҖӮдёҖиҲ¬дёҖдёӘз¬¬дёүж–№зҡ„еә”иҜҘиҝҷж ·е‘ҪеҗҚпјҢеғҸmybatisзҡ„mybatis-spring-boot-starterгҖӮ

  1. SpringBootжҠҖжңҜзұ»еҗҜеҠЁеҷЁ
