什么是项目？What is project?

The Project Management Institute defines it as a

“Temporary endeavor undertaken to create a unique product service or result”

换句话说： 它是一个临时性的举措，是为了实现一个特定目标而商定的计划和执行

项目是为完成某一独特的产品和服务所做的一次性努力。如果不是一次性，那就是运营（Operation）。

例子：

  • 建造一座工厂
  • 长城、金字塔
  • 组织一次旅游、家庭聚会

什么是项目管理？

项目管理就是运用各种知识、技能、手段和技术到项目活动中，以达到项目要求

硬技能

软技能

  • 情商
  • 沟通能力

项目管理的三重约束

时间范围和成本是相互依赖的，一旦确定它们之后， 它们中的一个发生变化，就会引发其他一个或两个的变化

Time

Temporary (complex) initiative 临时性举措

有一个预先的开始和结束日期

Goal

Specific goal you wish to achieve with the project

You can think of it as the output of a project

The output together with the work done to create it is called Project Scope. The scope of each project is unique

如果工作和产出是一样的，那么这将不是一个项目，项目范围不会是统一的，它将是一个常规的商业流程

Cost

The resources specifically dedicated to accomplishing the goal

项目管理的五大步骤

  • 启动

可行性研究、投资融资、立项授权、目标确立

  • 计划

造价预算、工期进度、技术方案设计、人员资源需求

  • 执行

采购管理、施工管理、技术管理、

  • 监控

范围监控、进度控制、质量控制、成本控制、人员管控、风险监控、变更与索赔、项目监理

  • 收尾

验收、移交/上线、文件归档、项目后评价

Why do companies execute projects?

Corporation:

  1. Daily work goals
  2. Stategic goals

How do companies transform these extra efforts into the desired benefits and goals?

Projects

A project must relate to a BUSINESS STRATEGY

Resources:

  • Financial resources

  • Organizational resources

  • Time

Benefits:

  • Financial
  • Operational
  • Customer related
  • Non-profit

What create the demand of a project in the first place?

Key triggers that will jump start a business into executing a project:

  • Marked Need
  • Business Need
  • Technological Advancement
  • Customer Request
  • Legal Requirements
  • Social Needs
  • Ecological Considerations

Urgency is a key factor when selecting projects

Project selection

Essential task for top managerment

In the eyes of the board of directors, projects are investment:

They use resources again, financial and otherwise, with the objective of seeing a worth while return

What is a project manager?

The project managerment is the CEO of the project

They will be accountable for the project’s success

What if the time frame and budget are insufficient?

A project manager can’t shoot themselves in the foot and take on a project that is unlikely to be successful.

Is the project manager’s duty to assess a project and deem the constraints practical or otherwise and if otherwise, they must demonstrate why to whoever proposed and negotiate for additional time, resources or an adjustment to the goal itself.

What does it mean to be accountable?

Project maganers are accountable not only for their own tasks but also all the tasks that other parties must work on and complete

What are the skills and the knowledge a project management should have?

Project manager: The face of the project & The point of contact

  1. Technical skills and knowledge

  2. Interpersonal(Soft) skills

  3. Industry experience

Project Management Office

This is the name of the department responsible for managing, coordinating and consulting project related work.

Phases of project development

  1. Initiation

  • They will define the overall objective or concept or the problem which needs resolving.

  • They will suggest a budget and the time frame and put all of this in the project business case and public service announcement for all aspiring project managers.

  • This is also the phase where the project manager is chosen.

  1. Planning

  • The project manager needs to set the work and organize how the project team will execute it. They estimeate and define resources and risks. Research is key.

  • The project manager must ask the right questions and work with other project stakeholders to find the answers. They need to find out things like what activities need to be performed to create the deliverables and project scope, how much time should be given to each one of these activities, what is the required budget, what activities will each member of the project team do and plenty more.

  • Finally, the project manager must document all of these findings into a project management plan

    The Recipe for how to execute the work to achieve the goal. The better this recipe is more comprehensive, detailed, realistic, the higher the project’s chances of being a delicious success.

planing=strategy for winning

  1. Execution
  2. Monitering and control

  • This phase runs in parallel to Execution

  • The project manager performs monitoring and control activities they have developed to track the progress of the strams and ask

    • Is the execution progressing as per the plan we created?
    • Is the speed and quality in line with the timeframe?
    • Are the expenses within the budget?

    If not, if there are delays, slowdowns or gaps, the project manager must identify them and take action to fix them before it’s too late.

  1. Closure phase
  • They must hand the project over to whomever it was designed for
  • The sponsor must officially approve and sign off the project confirming the agreed objective has been achieved.
  • Then the project team, the stakeholders and the project manager sit and discuss the session learnt.
    • What went well & what didn’t
    • What can be improved for future projects
